# Canary LayerZero DVN Statistics

A comprehensive Dune Analytics dashboard tracking Canary Decentralized Verifier Network (DVN) activity and statistics for the LayerZero protocol.

## Overview

This dashboard tracks cross-chain transactions on LayerZero verified by Canary DVN, showing the number of transactions and total value transferred. These metrics represent the total volume secured by Canary DVN.

## Data Sources

All data is queried directly from on-chain smart contract interactions across LayerZero-supported blockchains using Dune's SQL analytics platform.
